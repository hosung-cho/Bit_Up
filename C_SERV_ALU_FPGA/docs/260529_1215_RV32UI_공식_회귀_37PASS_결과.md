# RV32UI 공식 회귀 37 PASS 결과

## 요약

`riscv-tests` RV32UI 공식 테스트 37개가 PYNQ-Z2 FPGA host + 1-bit serial ASIC execution unit 구조에서 모두 통과했다.

## 실행 조건

- 위치: `testbench/4_fpga_host_riscv_tests/sim`
- 빌드: `./build_compliance.sh`
- 회귀: `./run_compliance.sh`
- 실행 환경: Vivado/xsim sandbox 밖 실행

Codex sandbox 내부에서는 xsim simulate 단계에서 `unexpected exception`이 발생할 수 있으므로, Vivado 검증은 sandbox 밖 실행 결과를 기준으로 한다.

## 최종 결과

- TOTAL: 37
- PASS: 37
- FAIL: 0
- ELAPSED: 501s

통과한 테스트:

`add addi and andi auipc beq bge bgeu blt bltu bne jal jalr lb lbu lh lhu lui lw or ori sb sh sll slli slt slti sltiu sltu sra srai srl srli sub sw xor xori`

## 수정 내용

- compliance harness의 data memory 크기를 공식 load/store 테스트가 접근하는 주소 범위에 맞게 `2048` bytes로 복구했다.
- pass/fail mailbox를 `0x700`으로 복구했다.
- 공식 load/store 테스트의 `.data` section은 `0x300` 이후에 배치되며, `lw`는 `0x380` 주소를 사용한다. 따라서 `DMEM_BYTES=512` 설정은 부적절했다.

## 의미

- 1-bit serial ASIC execution unit과 FPGA host bridge는 RV32I ALU/branch/jump/load/store directed path뿐 아니라 공식 RV32UI 테스트에서도 기능적으로 일관됨을 확인했다.
- 다음 단계는 PYNQ-Z2 wrapper/XDC와 MPW 28핀 ASIC 인터페이스를 실제 board bring-up 구조로 묶는 것이다.
